# 🐺 Tomiris Core

The central intelligence and modular engine of the **Tomiris AI Agent** ecosystem, engineered under the **DeltaVerse** architecture. 

Tomiris is designed to be a lightweight, lightning-fast, and completely modular AI assistant running locally on Linux systems.

---

## 🛠️ Key Features
* **Modular Architecture:** Dynamic plugin/module loading system (`modules/`).
* **GitHub API Integration:** Automated, secure, and token-free (or token-based) over-the-air module updates via SSH.
* **Native Localization:** Built-in core translation matrix providing **100% Turkish interface output**, regardless of backend module languages.
* **Open Source:** Licensed under the permissive MIT License.

## 📁 Repository Structure
* `src/` - Core engine source code.
* `modules/` - Hot-swappable functional modules (Speech, Vision, System Control).
* `config/` - Local environment variables and secure API configurations.
* `scripts/` - Automated installation and deployment tools.
